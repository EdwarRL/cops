# cops
The purpose of this model is to represent the Colombian power system in a mathematical optimization model in order to evaluate the realiability and the resilence with the integration of renewable energy resources to the system.

python -m venv .venv

.venv\Scripts\Activate.ps1

pip install -r requirements.txt

python main.py
